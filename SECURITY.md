# Security Policy

## Reporting a Vulnerability

Please report security issues to: security@example.com

Do NOT create GitHub issues for security vulnerabilities.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 5.x     | ✅ Yes             |
| 4.x     | ❌ No              |
| < 4.0   | ❌ No              |
